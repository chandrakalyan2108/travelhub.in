About TravelHub

TravelHub is a demo multi-category travel booking platform built with Java, Spring Boot and Thymeleaf. It brings together hotels, flights, trains, buses, movie tickets and metro passes in a single booking and payment flow, packaged as a WAR for deployment on Apache Tomcat 9.

This is a portfolio / learning project. Payments are simulated (no real payment gateway is wired up) — see the README for how to plug in Razorpay or Stripe.
